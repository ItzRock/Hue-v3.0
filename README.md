## About The Project

Hue is a discord bot based orginally for the roblox group foliage productions and now is going under its 3rd revision.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ItzRock/Hue-v3.0.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
